Контейнер для формы 
<form></form>

Атрибуты для верстальшиков
action - куда оптравлять форму
method - тип отправки формы

Теги формы
  input - основной тег, его режим работы настраиватся атриубтом type
      атрибуты:
        type - тип инпута, влияет на поведение конкретного инпута
        autofocus - фокусируется при переходе на страницу
        name - именует данные при их передаче
        required - обязательность данных в инпуте
        autocomplete - автозаполнение для конкретного элемента
        pattern - шаблон регулярных выражений для сверки с введенным текстом.
        Работает для type: text, email, password, url, tel, search


  label - тег для указывать назначение элемента формы, связывается с элементом 
    посредством атрибута for в который кидаем id эелемента, либо засовывая элемент в label

  button type="submit - кнопкя дял отправки формы

  fieldset - группировка элементов по смылу в формах на подобии section и article
    legend - элемент для описания названия группы fieldset

  
Типы инпутов:

  Основные:
    - text
    - email
    - password
    - radio
    - checkbox

  Второстепенные:
    - date
    - datetime
    - datetime-local
    - submit
    - button
    - reset
    - file
    - color
